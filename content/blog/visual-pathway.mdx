ヒトは目で見ているように思われがちですが、実際には「見ている」のは**脳**です。目から入った情報は、脳内の「視覚経路（Visual Pathway）」を通って処理され、私たちが世界を認識できるようになります。

この記事では、視覚情報がどのようなルートで脳内を通過し、どのように「意味あるイメージ」になるのかをやさしく見ていきます。

---

## 視覚の出発点：網膜から視神経へ

目に入った光は、まず網膜（retina）に届きます。
網膜には光を感じる\*\*視細胞（光受容体）\*\*があり、ここで光が電気信号に変換されます。

この電気信号は、\*\*視神経（optic nerve）\*\*を通って脳へと送られます。

```plaintext
光 → 網膜（視細胞）→ 電気信号 → 視神経 → 脳へ
```

---

## 交差する視覚情報：視交差（optic chiasm）

視神経は脳の「視交差」という場所で交差します。

> 右目の視野の情報は左脳へ、左目の視野の情報は右脳へと送られるのです。

これは、脳が\*\*対側性（contralateral）\*\*に働くという性質によるものです。

---

## 情報の中継地点：外側膝状体（LGN）

視交差を通過した信号は、\*\*視床の外側膝状体（lateral geniculate nucleus, LGN）\*\*という中継地点にたどり着きます。

ここでは情報が整理・統合され、次の目的地「一次視覚野（V1）」に送られます。

## 脳の中の視覚処理センター：一次視覚野（V1）

一次視覚野（Visual Cortex V1）は、後頭葉にあります。
ここで視覚の\*\*基本的な特徴（線、縁、方向）\*\*が抽出されます。

```js
// 脳内での視覚処理の例（イメージ）
const visualInfo = detectEdges(image);
const lines = detectOrientation(visualInfo);
const shapes = integrateFeatures(lines);
```

---

## ２つの経路：腹側路と背側路

一次視覚野からの情報は、以下の２つの経路に分かれて処理されます：

1. **腹側路（ventral pathway）**

   * 別名「**What経路**」
   * 形、色、顔などを識別
   * 側頭葉へ向かう

2. **背側路（dorsal pathway）**

   * 別名「**Where経路**」または「How経路」
   * 空間的位置、動き、行動計画に関与
   * 頭頂葉へ向かう

---

## まとめ：私たちはどうやって「見る」のか？

視覚とは単なる「映像」ではなく、以下のように複雑な処理の積み重ねで成り立っています。

1. 網膜で光が電気信号に変換される
2. 視神経 → 視交差 → LGN → V1へと信号が伝達
3. 一次視覚野で基本情報を処理
4. それが「腹側路」と「背側路」で別々の意味付け処理をされる

> 脳の中では、見ること＝理解すること なのです。
